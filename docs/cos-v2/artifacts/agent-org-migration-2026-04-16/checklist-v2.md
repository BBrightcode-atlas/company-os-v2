# 운영원칙체크리스트

1. **VERIFY** — 모든 산출물은 독립 검증 단계를 거쳐야 한다.
2. **TIMEOUT** — 각 임무는 제한 시간 안에 완료해야 한다.
3. **SCOPE** — 할당된 범위 밖의 파일은 수정하지 않는다.
4. **RETRY** — 실패한 단계는 원인 분석 뒤에 재시도한다.
5. **OWNER** — 담당자를 명시하여 책임 소재를 분명히 한다.
6. **LOG** — 주요 행동과 근거를 모두 기록으로 남긴다.
7. **BUDGET** — 예산 한계를 초과하기 전에 상위에 보고한다.
